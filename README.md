# Rigo Consulting website

This repository contains the static website for **Rigo Consulting AB**.

It is a small, single-page site built with plain HTML and CSS, and is deployed via GitHub Pages at:

https://kvarak.github.io/rigoconsulting/

The content presents my consulting work (continuous delivery, platform engineering and developer experience), my connection to [Tabete](https://tabete.se/) and a few side projects.

## Development

There is no build step. Open `index.html` directly in a browser or serve the folder with any simple static file server.
